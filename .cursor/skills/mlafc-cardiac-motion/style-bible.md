# Midnight Atlas, cardiac diagram style bible

## Palette (mandatory)

| Token | Hex | Use in diagrams |
|---|---|---|
| paper | #F7F5F1 | Background |
| ink | #122B3A | Heart outline, labels, primary lines |
| ink-soft | #3A5468 | Secondary annotations |
| ink-mute | #6B7F8E | Grid, ticks, de-emphasised paths |
| brass | #B08D3E | ECG pulse, active pathway, highlights |
| line | #D9D3C7 | Hairline rules |

## Motifs (from site identity)

- Dotted arc (London–Mumbai / logo ECG echo)
- Hairline cartographic grid at 5–10% opacity
- Coordinate tick marks at diagram corners
- ECG strip along bottom third
- Simple heart outline (4 chambers suggested, not anatomical cross-section)

## Motion language (Seedance)

- Slow push-in ≤5%
- Brass pulse travelling along ECG path (1–2 second cycle)
- Flicker → settle for arrhythmia → rhythm transitions
- No bounce, no parallax, no particle effects
- Loop-friendly: end frame ≈ start frame

## Forbidden visuals

Photorealism, surgery, blood, catheters, pacemaker surgery photos,
X-rays, MRI slices, hospital rooms, doctors' hands, patient faces,
dramatic lighting, lens flare, stock-photo aesthetic.
